complete k-partite graph


A complete k-partite graph is a maximal k-partite graph. In other words, a graph is complete k-partite provided that its vertex set admits a partition V=V1⊔…⊔Vk such that for any u∈Vi and v∈Vj, uv is an edge if and only if i≠j.

For any tuple (a1,…,ak), there is up to graph isomorphism a single complete k-partite graph with maximal stable sets V1,…,Vk such that for each i, there are exactly ai vertices in Vi. This graph is denoted by Ka1,…,ak.
