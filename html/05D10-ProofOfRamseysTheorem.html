proof of Ramsey’s theorem


ω→(ω)kn

is proven by induction on n.

If n=1 then this just states that any partition of an infinite set into a finite number of subsets must include an infinite set; that is, the union of a finite number of finite sets is finite. This is simple enough to prove: since there are a finite number of sets, there is a largest set of size x. Let the number of sets be y. Then the size of the union is no more than x⁢y.

If

ω→(ω)kn

then we can show that

ω→(ω)kn+1

Let f be some coloring of [S]n+1 by k where S is an infinite subset of ω. Observe that, given an x<ω, we can define fx:[S∖{x}]n→k by fx⁢(X)=f⁢({x}∪X). Since S is infinite, by the induction hypothesis this will have an infinite homogeneous set.

Then we define a sequence of integers ⟨ni⟩i∈ω and a sequence of infinite subsets of ω, ⟨Si⟩i∈ω by induction. Let n0=0 and let S0=ω. Given ni and Si for i≤j we can define Sj as an infinite homogeneous set for fni:[Sj-1]n→k and nj as the least element of Sj.

Obviously N=⋃{ni} is infinite, and it is also homogeneous, since each ni is contained in Sj for each j≤i.
