A \emph{pseudograph} is a graph that allows both parallel edges and loops.
Formally, $G=(V, E)$ is a pseudograph , if $E$ is a multiset $(V^{(2)}, f)$
where $V^{(2)}$ is the set of unordered pairs of $V$.
